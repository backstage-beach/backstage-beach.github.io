## About

Backstage Beach is an AWS-focused developer portal built on Backstage, providing:

- Service catalog for AWS resources
- Software templates for rapid project creation
- Technical documentation and best practices
- Self-service infrastructure provisioning

## Development

This site is built with GitHub Pages.

```bash
# Clone the repository
git clone https://github.com/backstage-beach/backstage-beach.github.io.git
cd backstage-beach.github.io

# Serve locally (if using Jekyll)
bundle install
bundle exec jekyll serve
```
